Task Manager API (Flask + SQLite)

Task Manager API

Мини‑backend для управления задачами с использованием Flask и SQLite.

---

📌 Описание проекта

Task Manager API — учебный REST‑сервис, позволяющий создавать, удалять и просматривать задачи. Проект демонстрирует базовые навыки работы с базой данных, CRUD‑операциями и построением API.

---

🔧 Стек технологий

• Python
• Flask
• SQLite
• SQLAlchemy
• Git / GitHub


---

🚀 Функционал

• POST /tasks — создание задачи
• GET /tasks — получение всех задач
• DELETE /tasks/<id> — удаление задачи
• Валидация входящих данных
• ORM‑модели SQLAlchemy


---

📁 Структура проекта

task-manager-api/

│── models.py      # ORM-модели
│── database.py    # Подключение к базе

│── README.md


---

▶️ Запуск проекта

Установка зависимостей:

pip install -r requirements.txt


Запуск сервера:

python app.py
